# PPT 分享页生成规范

## 1. 核心概念

这是一个全屏翻页演示页面，类似 PPT/reveal.js，但完全自包含。目标受众是开发人员，内容是源码架构分析的精华，不是学习参考文档。

**定位区别：**
| 维度 | 学习页面 | 分享页面 |
|------|---------|---------|
| 形式 | 长页面滚动 | 全屏翻页 |
| 内容 | 详尽参考 | 精华提炼 |
| 阅读方式 | 查阅式 | 演示式 |
| 目标 | 深度理解 | 快速传达 |

## 2. 设计系统

### 配色
- 背景: #0a0e1a (深色午夜蓝)
- 主色: #00d4ff (cyan)
- 辅色: #7c3aed (purple)
- CTA色: #f97316 (orange)
- 文字: #e2e8f0
- 卡片: rgba(15,23,42,0.8) + backdrop-blur

### 字体
- 标题/代码: JetBrains Mono
- 正文: IBM Plex Sans

### 效果
- 玻璃拟态卡片 (backdrop-filter: blur)
- 微妙的辉光效果 (box-shadow with accent color)
- 渐变背景动画 (subtle, 不分散注意力)

## 3. 幻灯片引擎

### 核心机制
- 每页 100vh 全屏
- 容器用 `transform: translateY(-${n * 100}vh)` 切换
- 过渡: `transition: transform 500ms cubic-bezier(0.4, 0, 0.2, 1)`

### 导航方式
| 方式 | 操作 | 说明 |
|------|------|------|
| 键盘 | ↑↓←→ 和 Space | 上一页/下一页 |
| 滚轮 | 鼠标滚动 | 防抖 700ms |
| 触屏 | 上下滑动 | 阈值 40px |
| 指示器 | 右侧圆点 | 点击跳转 |
| 概览 | Esc 键 | 缩略图网格 |

### 必须的 UI 元素
1. **右侧页面指示器** — 垂直排列的圆点，当前页高亮
2. **页码计数器** — 右下角 "N / Total"
3. **阅读进度条** — 顶部细条
4. **导航提示** — 首页显示 "按方向键翻页"

### 概览模式 (Esc)
- 所有幻灯片缩小为 4 列网格
- 点击任一缩略图跳转
- 再按 Esc 退出概览

## 4. 幻灯片结构 (15-18 页)

### 推荐结构

| 页码 | 类型 | 内容 |
|------|------|------|
| 1 | 封面 | 项目名 + 副标题 + 操作提示 |
| 2 | 介绍 | 项目是什么 + 技术栈 |
| 3 | 数据 | 关键数字（代码量/工具数/命令数）+ 动画计数器 |
| 4 | 架构 | SVG 全局架构图 |
| 5-6 | 启动 | Boot sequence / 初始化流程 |
| 7-9 | 核心 A | 项目最核心的机制（2-3 页） |
| 10-12 | 核心 B | 第二核心机制（2-3 页） |
| 13-14 | 扩展 | 插件/配置/扩展体系 |
| 15-16 | 哲学 | 设计原则和 trade-off |
| 17 | 总结 | 关键 takeaway |
| 18 | 结尾 | 感谢 + 链接 |

### 每页内容要求（CRITICAL）
- **不是概要卡片** — 每页必须有真实内容（代码、图、详细解释）
- 代码块不超过 15 行（精选最关键的部分）
- SVG 图简洁清晰，不要过度复杂
- 文字精炼但有信息量 — 不是空洞的标题

### 单页布局模式

| 模式 | 适用场景 | 布局 |
|------|---------|------|
| 居中标题 | 封面/结尾 | flex-center, 大字号 |
| 双栏 | 代码+解释 | grid 2-col, 左代码右文字 |
| 卡片网格 | 多概念并列 | grid 2x2 或 3x2 |
| 全图 | 架构图 | SVG 居中 + 底部说明 |
| 层叠 | 优先级/层级 | 垂直堆叠的色带 |

## 5. 质量标准

### 视觉
- 每页有视觉焦点（图/代码/大数字/卡片）
- 不要纯文字页（至少有一个视觉元素）
- 颜色统一使用设计系统变量
- 不用 emoji 做 icon，用 SVG

### 交互
- 所有可点击元素有 cursor-pointer
- 过渡动画 200-300ms
- 页面切换 500ms
- prefers-reduced-motion 时禁用动画

### 内容
- 中文文字，英文代码/术语
- 代码用 Prism.js 高亮
- 每页阅读时间 30-60 秒
